FoodHub is a backend-based food ordering application developed using **Node.js, Express.js, and MongoDB** to manage food ordering operations efficiently. The project implements both **Server-Side Rendering (SSR)** using EJS templates and **Client-Side Rendering (CSR)** through static pages. MongoDB with Mongoose is used for database management and performing CRUD operations. Middleware concepts including application-level, router-level, error handling, and third-party middleware such as Helmet, CORS, and Morgan are integrated for request processing and security. Authentication and session management are implemented using **BcryptJS, JWT, PassportJS, Express-Session, and Cookies** to provide secure user access and login management. The project also includes **Socket.IO** for real-time full duplex communication and follows a structured backend architecture for handling routes, APIs, authentication, and database connectivity.
